# Wdrożenie BitLocker w sieci komputerów z OS Windows 11 z modułem TMP
### autor: Michał 182954

W przypadku pracy z Danymi Wrażliwymi, niezbędne jest zadbanie o bezpieczeństwo dostępu do tych danych. 
Funkcją która wspiera ograniczenie dostępu do danych w przypadku posiadania fizycznego dostępu do urządzenia jest szyfrowanie dysku, a jednym z prostszych narzędzi to tego jest Windows BitLocker. 
Projekt przedstawia wdrożenie tego narzędzia w sieci opartej o Active Directory dla komputerów Windows 11 z modułem TMP 2.0
